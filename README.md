# devenv-doctor
DevEnv Doctor is a CLI tool that analyzes Docker and Docker Compose projects to determine whether they are ready to run in a local development environment. It checks the project against predefined operational policies and best practices, calculates a readiness score, and provides actionable recommendations to fix detected issues.
